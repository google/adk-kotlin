# Examples

> Internal. Part of the ADK Kotlin specification: the repo-root `SPEC.md` is the
> charter and index, and `DECISIONS.md` holds the `[D-n]` design decisions. Not
> exported to GitHub.

`examples` (JVM) and `examples/android` are runnable demos (not published). They
cover: hello, A2A consumer, GCS artifacts, callbacks/observers, event
compaction, GitHub tools + a docs-release analyzer, HITL (confirmation and
helper tools), on-device LiteRT-LM, skills, structural agents
(sequential/loop/parallel), telemetry, agent-tool, function-tool (KSP), Google
Search, and agent transfer. Android: Room-backed session persistence and skills
from APK assets. New public capabilities should ship with a matching example.
